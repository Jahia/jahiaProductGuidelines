Buttons

There are three primary button types. All the css and javascript required are already included in the jahia_components.css and jahia_components.js files. Just copy the html where you want the button on your page.

Default
Icon button
Custom button

Default button

Default buttons are used across the product where an action from the user is possible. They can vary in width depending on the size of the name. You can also use custom colors for a CTA, just add a color class to your html.

HTML
Copy<button type="button" class="button_defaultButton" ripple="true">Button</button>
CSS
Copy.button_defaultButton {
	background: rgb(200, 200, 200);
	color: rgb(107,107,107);
	border: none;
	padding: 5px 10px;
	font-size: 18px;
	width: 150px;
	outline: none;
}

.button_defaultButton:hover {
	background: rgb(230, 230, 230);
	cursor: pointer;
}
info_outline The css above is for illustration purpose only, the jahia_components.css already contains everything you need.

Icon button

Icon buttons are used primarily for quick actions and for shortcuts like the ones you can find on the sidebars of DX8. You must choose an icon consistent with the action the button is supposed to perform. Refer to Style > Icons for examples.

HTML
Copy<button type="button" class="button_iconButton" ripple="true">
	
	<!-- Change icon here -->
	<i class="material-icons">perm_media</i>
	Button
</button>
